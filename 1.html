  • Library Assistant Bot (Partially Completed)

    • The bot identifies the correct location of the books placed at wrong locations in the library.

    The robot focus on mitigating the workload of the librarian by automatically sorting all the books in the library which are placed at wrong locations. The robot uses Raspberry Pi as the master microcontroller with Arduino as slave.

    Current Status: The bot incurred with a line follower which make sure to run on specified paths. The books contain RFID tags which are scanned by RFID reader present on the bot and matches its location with the current location ensuring the book is at the right place. The database of the books and their respectively locations is made in the python. The Floyd-Warshall Algorithm is used in order to get the shortest path to its destination.
